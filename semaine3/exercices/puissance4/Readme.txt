* Aucun de ces fichiers Java n'est terminé

* Si à jamais vous terminez le projet (cas du terminal de Ubuntu 22.04),
    - pour le compiler, il faudra se placer dans le repertoire parent du dossier puissance4
      et executer 'javac puissance4/*.java'
    - pour l'execution, il faudra executer 'java puissance4.Main'
